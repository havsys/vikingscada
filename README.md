### Install
```
$ npm install
```

## Manually lint
```
$ npm run lint
```
Anyway linter will be run after file changes. Check out linter rules [here](https://github.com/airbnb/javascript).

## Run
```
$ npm start
```
Project is running at [http://0.0.0.0:8080/](http://0.0.0.0:8080/)
